# Real Estate Agency Database

This project presents the design and implementation of a **relational database for a real estate agency**.
The database supports management of properties, clients, agents, listings, offers, inspections, and contracts.

## Features

* Normalized relational schema
* SQL scripts for creating and dropping database structures
* Sample data insertion scripts
* Example SQL queries and views for data analysis

## Files

* `create.sql` – creates database tables and constraints
* `insert.sql` – inserts sample data
* `queries.sql` – example analytical queries
* `creating_view.sql` – view definitions
* `drop.sql` – removes database objects
* `erd_rbd.pdf` – ER diagram of the database schema

## Technologies

* SQL
* Relational database design

## Note

This database is a fictional project created for educational purposes.
All data and entities are simulated and do not represent real people, properties, or companies.
